# zMath

> Gen-Z를 위한 수학 울렁증 극복 커뮤니티 — 게임과 챌린지로 수학을 다시 시작한다.

## 배경 및 목적

- **문제**: 수학에 어려움을 겪다 포기하는 Gen-Z 수포자가 증가하고 있음
- **기회**: 게임·챌린지·랭킹 등 Gen-Z에 친숙한 포맷으로 수학 개념을 재구성하면 학습 진입 장벽을 낮출 수 있음
- **기대 효과**: 수포자의 재입문 → 개념 이해 → 성취감 형성의 선순환

## 핵심 기능

| # | 기능 | 설명 | 우선순위 |
|---|------|------|---------|
| 1 | 게임으로 배우는 수학 개념 | 개념별 인터랙티브 미니게임; 시각화 중심 | 필수 |
| 2 | 도전! 챌린지 | 기간/레벨 단위 문제 챌린지; 클리어 시 보상 피드백 | 필수 |
| 3 | 랭킹 | 챌린지 점수·스트릭 기반 리더보드 | 필수 |

## 대상 사용자

- **주요**: 10대 중·고등학생 (수학 기초 부족, 흥미 회복 필요)
- **보조**: 20대 (개념 재학습, 수능 재도전 등)

## 기술적 범위

- **포함**
  - 수학 개념의 시각화 (도형, 그래프, 수직선 등 인터랙티브 렌더링)
  - 게임 인터랙션 (드래그, 클릭, 슬라이더 기반 조작)
  - 챌린지·랭킹 UI
- **제외**
  - 사운드·음향 효과
  - 복잡한 애니메이션·파티클 등 고비용 시각 효과
  - 로그인·회원가입·인증 프로세스
  - 백엔드 API / 실제 데이터 저장 (프론트 프로토타입 범위)

## 성공 기준

- 게임 1개 이상 플레이 완주 가능한 인터랙션 완성도
- 챌린지 → 랭킹 진입까지 끊김 없는 UX 플로우
- 수포자 페르소나 기준 "어렵지 않다"는 첫인상
